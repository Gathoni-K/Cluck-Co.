# Simple E-Commerce Website

A **modern, responsive e-commerce web application** built with **React**, **TypeScript**, **Tailwind CSS**, and **ShadCN UI**. This project demonstrates my ability to build clean, user-friendly interfaces with essential e-commerce functionality, ready for real-world applications.

## Features
- Browse products with dynamic rendering  
- Add and remove items from the cart  
- Responsive design for mobile, tablet, and desktop  
- Clean UI using ShadCN UI components and Tailwind CSS  

## Tech Stack
- **Frontend:** React, TypeScript  
- **Styling:** Tailwind CSS  
- **UI Components:** ShadCN UI  
- **Routing & State:** React Router, React Context API
